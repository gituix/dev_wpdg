== Description ==

[Jcarousel Post Slider] can display a list of posts as horizontal slider or vertical slider using jcarousel jquery plugin.

See demo on: http://blog.casanova.vn/jcarousel-post-slider-plugin/ 

Some features of this plugin:

*   Can limit number of post to display
*   Can select categories that you want to display post from it
*   Can orders post by date, title, number of comment, random
*   Can set animation for slide, can choose skin or design skin 



== Installation ==

Very easy to install this plugin.

1. Upload `Jcarousel Post Slider` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto Appearance -->Widgets and drag Jcarousel Post Slider to your sidebar and config it
